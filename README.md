# dotfiles

### Validate AWS CLI Installation

```
aws --version
```

### Login to AWS IAM Identity Center

```
aws configure sso --use-device-code
```

### Validate AWS CDK Installation


```
cdk version
npm list -g aws-cdk-lib
```

### Validate Rust Installation

```
rustc --version
```

### Validate Cargo Installation

```
cargo --version
```
